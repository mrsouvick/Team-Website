How to Set Up ROS Noetic on Ubuntu

This guide will walk you through the steps to install and set up ROS Noetic on Ubuntu 20.04.

Step 1: Update Your Ubuntu System

Before installing ROS, ensure your system is up to date:

sudo apt update
sudo apt upgrade

Step 2: Set Up ROS Noetic Repository

Add the ROS Noetic repository to your system:

sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

Step 3: Add the ROS Key

Add the ROS key to authenticate the packages:

sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

Step 4: Install ROS Noetic

Update the package list and install ROS Noetic:

sudo apt update
sudo apt install ros-noetic-desktop-full

Step 5: Initialize rosdep

Initialize rosdep to manage dependencies:

sudo rosdep init
rosdep update

Step 6: Set Up Environment Variables

Add ROS environment variables to your shell startup script:

echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc
source ~/.bashrc

Step 7: Verify Installation

Test your ROS installation by running the following command:

roscore

If ROS is installed correctly, you should see output indicating that the ROS master is running.

Step 8: Install Additional Tools (Optional)

Install useful ROS tools like rosinstall and catkin:

sudo apt install python3-rosinstall python3-rosinstall-generator python3-wstool build-essential

Conclusion

Congratulations! You have successfully set up ROS Noetic on Ubuntu. You can now start developing and running ROS applications.

For more information, visit the official ROS Noetic Installation Guide.
